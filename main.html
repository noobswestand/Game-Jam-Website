Welcome to the Game Jam!
A game jam is a competition where you have a set amount of time to make a game based on a theme/set of rules/restrictions.

You have 21 days to put together a game based off of whatever you like. Extra points for the theme:

The Jam will be capped by a pizza party for all entrants!

If you do not know how to begin creating games, check out the resourse tab found here


Voting

  • Theme
    • How well the game fits the theme
  • Concept
    • The originality and how well the game is put together
    • This could be how well the game plays, and game feel
  • Presentation
    • Art, Music, Sound effects
  • Story
    • How interesting the story is. Does it fit?

Prizes

  • Your game will be featured on the front page of this website until the next game jam
  • If you want to contribute a prize, please email me: djk164@yahoo.com
